The FreeCADGui module¶

The functions in the FreeCADGui module allow working with GUI documents, view providers, views, workbenches and much more.

The FreeCADGui instance provides a list of references of GUI documents which can be addressed by a string. These documents contain the view providers for objects in the associated App document. An App and GUI document can be accessed with the same name.

The FreeCADGui module also provides a set of functions to work with so called workbenches.
