clear the sketch

movePoint(GeoIndex,PointPos,Vector,[relative]) - move a given point (or curve)¶

to another location. It moves the specified point (or curve) to the given location by adding some temporary weak constraints and solve the sketch. This method is mostly used to allow the user to drag some portions of the sketch in real time by e.g. the mouse and it works only for underconstrained portions of the sketch. The argument ‘relative’, if present, states if the new location is given relatively to the current one.
