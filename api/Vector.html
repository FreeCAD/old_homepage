The Vector object¶

The functions in the FreeCAD module allow working with documents. The FreeCAD instance provides a list of references of documents which can be addressed by a string. Hence the document name must be unique.

The document has the read-only attribute FileName which points to the file the document should be stored to.

class Vector¶

This class represents a 3D float vector

Length¶

Length([Float]) -> Float gets or sets the length of this vector

add(Vector)¶

returns the sum of this and another vector

cross(Vector)¶

returns the cross product between this and another vector

distanceToLine(Vector, Vector)¶

returns the distance between this vector and a line defined by a base point and a direction

distanceToLineSegment(Vector, Vector)¶

returns the distance between this vector and a line segment defined by a base point and a direction

distanceToPlane(Vector, Vector)¶

returns the distance between this vector and a plane defined by a base point and a normal

dot(Vector)¶

returns the dot product of the this vector with another one

getAngle(Vector)¶

returns the angle in radians between this and another vector

multiply(Float)¶

multiplies (scales) this vector by a single factor

normalize()¶

normalizes the vector to the length of 1.0

projectToLine(Vector, Vector)¶

projects the vector on a line defined by a base point and a direction

projectToPlane(Vector, Vector)¶

projects the vector on a plane defined by a base point and a normal

scale(Float, Float, Float)¶

scales (multiplies) this vector by a factor

sub(Vector)¶

returns the difference of this and another vector

x¶

x([Float]) -> Float gets or sets the X component of this vector

y¶

y([Float]) -> Float gets or sets the Y component of this vector

z¶

z([Float]) -> Float gets or sets the Z component of this vector
