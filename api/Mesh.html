The Mesh module¶

The functions in this module allow working with mesh objects. A set of functions are provided that allow to read in registered mesh file formats to either an newly created or already exising document.

open(string) – Create a new document and a Mesh::Import feature to load the file into the document. insert(string, string) – Create a Mesh::Import feature to load the file into the given document. mesh() – Create an empty mesh object.

class Facet¶

Facet in mesh This is a facet in a MeshObject. You can get it by e.g. iterating a mesh. The facet has a connection to its mesh and allows therefore topological operations. It is also possible to create an unbounded facet e.g. to create a mesh. In this case the topological operations will fail. The same is when you cut the bound to the mesh by calling unbound().

Bound¶

Bound state of the facet

Index¶

The index of this facet in the MeshObject

NeighbourIndices¶

The index tuple of neighbour facets of the mesh this facet is adjacent with

Normal¶

Normal vector of the facet.

PointIndices¶

The index tuple of point vertices of the mesh this facet is built of

Points¶

A list of points of the facet

intersect(Facet) → list¶

Get a list of intersection points with another triangle.

unbound()¶

method unbound() Cut the connection to a MeshObject. The facet becomes free and is more or less a simple facet. After calling unbound() no topological operation will work!

class Feature¶

The Mesh::Feature class handles meshes. The Mesh.MeshFeature() function is for internal use only and cannot be used to create instances of this class. Therefore you must have a reference to a document, e.g. ‘d’ then you can create an instance with d.addObject(“Mesh::Feature”).

class MeshPoint¶

Point in mesh This is a point in a MeshObject. You can get it by e.g. iterating a mesh. The point has a connection to its mesh and allows therefore topological operations. It is also possible to create an unbounded mesh point e.g. to create a mesh. In this case the topological operations will fail. The same is when you cut the bound to the mesh by calling unbound().

Bound¶

Bound state of the point

Index¶

The index of this point in the MeshObject

Normal¶

Normal vector of the point computed by the surrounding mesh.

Vector¶

Vector of the point.

move()¶

method move(Vector) This method moves the point in the mesh along the given vector. This affects the geometry of the mesh. Be aware that after moving point(s) the mesh can have self intersections!

unbound()¶

method unbound() Cut the connection to a MeshObject. The point becomes free and is more or less a simple vector/point. After calling unbound() no topological operation will work!

x¶

The X component of the point. Setting this value also affects the mesh if this point is connected to it.

y¶

The Y component of the point. Setting this value also affects the mesh if this point is connected to it.

z¶

The Z component of the point. Setting this value also affects the mesh if this point is connected to it.
