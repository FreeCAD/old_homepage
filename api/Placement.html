The Placement object¶

The functions in the FreeCAD module allow working with documents. The FreeCAD instance provides a list of references of documents which can be addressed by a string. Hence the document name must be unique.

The document has the read-only attribute FileName which points to the file the document should be stored to.

class Placement¶

Placement A placement defines an orientation (rotation) and a position (base) in 3D space. It is used when no scaling or other distortion is needed.

The following constructors are supported: Placement() – empty constructor Placement(Placement) – copy constructor Placement(Matrix) – 4D matrix consisting of rotation and translation Placement(Base, Rotation) – define position and rotation Placement(Base, Rotation,Center) – define position and rotation with center Placement(Base, Axis, Angle) – define position and rotation

Base¶

Vector to the Base Position of the Placement

Rotation¶

Orientation of the placement expressed as rotation

Returns a copy of this Placement

isNull() → Bool¶

Multiply this placement with another placement

convert the placement to a matrix representation
