Draft Vector Utilities¶

The DraftVecUtils module offer several convenient utilities to deal with vectors

angle(u, v=Vector (1, 0, 0), normal=Vector (0, 0, 1))¶

angle(Vector,[Vector],[Vector]) - returns the angle in radians between the two vectors. If only one is given, angle is between the vector and the horizontal East direction. If a third vector is given, it is the normal used to determine the sign of the angle.

dist(u, v)¶

dist(Vector,Vector) - returns the distance between both points/vectors

equals(u, v)¶

returns True if vectors differ by less than precision (from ParamGet), elementwise

find(vector, vlist)¶

find(vector,vlist): finds a vector in a list of vectors. returns the index of the matching vector, or None if none is found.

getPlaneRotation(u, v, w=None)¶

returns a rotation matrix defining the (u,v,w) coordinates system

isColinear(vlist)¶

isColinear(list_of_vectors): checks if vectors in given list are colinear

isNull(vector)¶

isNull(vector): Tests if a vector is nul vector

neg(u)¶

neg(Vector) - returns an opposite (negative) vector

project(u, v)¶

project(Vector,Vector): projects the first vector onto the second one

removeDoubles(vlist)¶

removes consecutive doubles from a list of vectors

rotate(u, angle, axis=Vector (0, 0, 1))¶

rotate(Vector,Float,axis=Vector): rotates the first Vector around the given axis, at the given angle. If axis is omitted, the rotation is made on the xy plane.

rotate2D(u, angle)¶

rotate2D(Vector,angle): rotates the given vector around the Z axis

rounded(v)¶

returns a rounded vector

scale(u, scalar)¶

scale(Vector,Float) - scales (multiplies) a vector by a factor

scaleTo(u, l)¶

scaleTo(Vector,length) - scales a vector to a given length

toString(u)¶

returns a string containing a python command to recreate this vector

tup(u, array=False)¶

returns a tuple (x,y,z) with the vector coords, or an array if array=true
