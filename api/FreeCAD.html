The FreeCAD module¶

The functions in the FreeCAD module allow working with documents. The FreeCAD instance provides a list of references of documents which can be addressed by a string. Hence the document name must be unique.

The document has the read-only attribute FileName which points to the file the document should be stored to.

ConfigGet([string]) – Get the value for the given key. If no key is given the complete configuration is dumped to the output.

getResourceDir()¶

Get the root directory of all resources

listDocuments() → list¶

Return a list of names of all documents.

loadFile(string=filename[, string=module]) → None¶

Loads an arbitrary file by delegating to the given Python module: * If no module is given it will be determined by the file extension. * If more than one module can load a file the first one one will be taken. * If no module exists to load the file an exception will be raised.

newDocument([string]) → object¶

Create a new document with a given name. The document name must be unique which is checked automatically.

open()¶

See openDocument(string)

openDocument(string) → object¶

Create a document and load the project file into the document. The string argument must point to an existing file. If the file doesn’t exist or the file cannot be loaded an I/O exception is thrown. In this case the document is kept alive.

removeDocumentObserver() → None¶

Remove an added document observer.
