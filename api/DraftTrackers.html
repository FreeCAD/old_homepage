Draft Trackers¶

Different trackers (temporary screen widgets) used in the Draft and Arch modules.

class PlaneTracker¶

A working plane tracker

class Tracker(dotted=False, scolor=None, swidth=None, children=, []ontop=False)¶

A generic Draft Tracker, to be used by other specific trackers

lowerTracker()¶

lowers the tracker to the bottom of the scenegraph, so it doesn’t obscure the other objects

raiseTracker()¶

raises the tracker to the top of the scenegraph, so it obscures the other objects

class arcTracker(dotted=False, scolor=None, swidth=None, start=0, end=6.283185307179586, normal=None)¶

An arc tracker

getAngle(pt)¶

returns the angle of a given vector

getAngles()¶

returns the start and end angles

getDeviation()¶

returns a deviation vector that represents the base of the circle

getRadius()¶

returns the current radius

setApertureAngle(ang)¶

sets the end angle by giving the aperture angle

setCenter(cen)¶

sets the center point

setEndAngle(ang)¶

sets the end angle

setEndPoint(pt)¶

sets the end angle from a point

setRadius(rad)¶

sets the radius

setStartAngle(ang)¶

sets the start angle

setStartPoint(pt)¶

sets the start angle from a point

class archDimTracker(p1=Vector (0.0, 0.0, 0.0), p2=Vector (1.0, 0.0, 0.0), mode=1)¶

A wrapper around a Sketcher dim

p1(point=None)¶

sets or gets the first point of the dim

p2(point=None)¶

sets or gets the second point of the dim

setMode(mode=1)¶

sets the mode: 0 = without lines (a simple mark), 1 = aligned (default), 2 = horizontal, 3 = vertical.

setString(text=None)¶

sets the dim string to the given value or auto value

class bezcurveTracker(dotted=False, scolor=None, swidth=None, points=[])¶

A bezcurve tracker

class boxTracker(line=None, width=0.1, height=1)¶

A box tracker, can be based on a line object

class bsplineTracker(dotted=False, scolor=None, swidth=None, points=[])¶

A bspline tracker

class dimTracker(dotted=False, scolor=None, swidth=None)¶

A Dimension tracker, used by the dimension tool

class editTracker(pos=Vector (0.0, 0.0, 0.0), name='None', idx=0, objcol=None, marker=81)¶

A node edit tracker

class ghostTracker(sel)¶

A Ghost tracker, that allows to copy whole object representations. You can pass it an object or a list of objects, or a shape.

center(point)¶

sets the rotation/scale center of the ghost

getNode(obj)¶

gets a coin node which is a full copy of the current representation

getNodeLight(shape)¶

extract a lighter version directly from a shape

move(delta)¶

moves the ghost to a given position, relative from its start position

rotate(axis, angle)¶

rotates the ghost of a given angle

scale(delta)¶

scales the ghost by the given factor

update(obj)¶

recreates the ghost from a new object

class gridTracker¶

A grid tracker

getClosestNode(point)¶

returns the closest node from the given point

class lineTracker(dotted=False, scolor=None, swidth=None)¶

A Line tracker, used by the tools that need to draw temporary lines

getLength()¶

returns the length of the line

p1(point=None)¶

sets or gets the first point of the line

p2(point=None)¶

sets or gets the second point of the line

class radiusTracker(position=Vector (0.0, 0.0, 0.0), radius=1)¶

A tracker that displays a transparent sphere to inicate a radius

class rectangleTracker(dotted=False, scolor=None, swidth=None, face=False)¶

A Rectangle tracker, used by the rectangle tool

getNormal()¶

returns the normal of the rectangle

getSize()¶

returns (length,width) of the rectangle

p1(point=None)¶

sets or gets the base point of the rectangle

p2()¶

gets the second point (on u axis) of the rectangle

p3(point=None)¶

sets or gets the opposite (diagonal) point of the rectangle

p4()¶

gets the fourth point (on v axis) of the rectangle

setPlane(u, v=None)¶

sets given (u,v) vectors as working plane. You can give only u and v will be deduced automatically given current workplane

setorigin(point)¶

sets the base point of the rectangle

update(point)¶

sets the opposite (diagonal) point of the rectangle

class snapTracker¶

A Snap Mark tracker, used by tools that support snapping

class wireTracker(wire)¶

A wire tracker
